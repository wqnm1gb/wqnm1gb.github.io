Day03 循环语句

一月 2, 2025 · 2 分钟 · 610 字 · 冷漠三狗子丶

if的注意点#

1、在语句中，如果只有一句代码，那么大括号可以省略不写

int a = 100;
//这不算是一句代码，算是两句代码，一句是定义变量a，另一句是赋值

2、如果对一个布尔类型的变量进行判断，不要使用==号，直接把变量写在小括号中

switch语句#

switch (表达式){
    case 值1:         //case后边的值只能是字面量，不能是变量
        语句体1;
        break;
    case 值2:
        语句体2;
        break;
    ...
    default:
        语句体n+1;
        break;
}

default可以省略，语法不会有问题，但是不建议省略，default可以放在程序的任意位置

case穿透，语句体中没有写break导致的

如果匹配上了，就会执行对应的语句体，如果此时发现了break，那么结束switch语句，如果没有发现break，那么程序就会继续执行下一个case语句，一直到遇到break或者右大括号为止

如果多个case的语句体重复了，那么我们考虑用case穿透去简化代码

if else一般是对范围的判断

switch 把有限个数据一一列举出来，让我们任选其一

循环语句#

100.fori 可以快速的写出循环100次的

for和while的区别#

for循环中，控制循环的变量（变量i），归属于for循环的语法结构中，在for循环结束之后，就不能再次访问了（但是改写之后，可以改变这个规则）

while循环中，控制循环的变量（变量i），不归属于while的循环结构中，在while循环结束之后，该变量还能继续访问

for循环一般知道循环的次数和范围

while循环不知道次数和范围，但是知道结束条件
