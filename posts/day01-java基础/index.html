Day01 Java基础

一月 2, 2025 · 1 分钟 · 412 字 · 冷漠三狗子丶

cmd默认操作的是用户命名的文件夹（C:\users\用户名）

盘符名称 + : 能够切换到这个盘符

dir 查询该目录下的所有文件

window下的文件分隔符是\ 注意与Linux下的/ 不同

cls 是清屏命令

exit 是关闭窗口命令

java目录 bin中存放了各种工具命令

JDK java的开发工具 里边包含了 JVM+核心类库+开发工具

JRE java的运行环境 里边包含了 JVM+核心类库+运行工具

多行注释是 /* */

文档注释是/** **/

Java会先找到main语句，从main开始运行

javac用于编译

java用于运行

类是Java最基本的组成单元

空类型的null不能直接打印

在打印的时候，把前边字符串的长度补齐到8，或者8的整数倍，最少补1个空格，最多补8个空格

用处：打印类似表格数据的时候，让数据对齐

2、变量名不允许被重复定义

3、一条语句可以定义多个变量

4、变量在使用之前一定要进行赋值

5、变量的作用范围

十进制：前边不加任何前缀

八进制：代码中以0开头

十六进制：代码中以0x开头
